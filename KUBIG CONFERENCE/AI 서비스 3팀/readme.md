## MYSNAP 소개
- 내가 원하는 컨셉, 분위기, 스타일에 맞는 작가를 추천해주는 서비스
- 기능 1: 분위기 중심 추천
  분위기 옵션 중 원하는 스타일을 선택하고 추가로 텍스트로 구체적인 요구 사항을 입력할 수 있는 맞춤형 추천 서비스 제공
- 기능 2 : 고정된 요구사항 기반 추천
  명확한 요구를 기반으로 필터링한 후 추가적인 무드나 배경을 자유롭게 입력해 맞춤 추천을 제공
